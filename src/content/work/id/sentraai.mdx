## Problem

*[KONFIRMASI BANI] — latar belakang kebutuhan pasar yang mendorong pembuatan produk chat AI konsumen ini belum tercantum di dokumen sumber; perlu konfirmasi untuk detail lebih lengkap.*

## Solution

Bani membangun **SentraAI**, layanan chat dengan AI assistant yang bisa diakses dari mana saja dengan interface aman dan dioptimasi untuk mobile. Produk berjalan dengan model subscription, menekankan keamanan (API key tersimpan di backend, bukan di browser) dan performa (response cepat lewat caching).

## Tools

- AI Chat Assistant lintas perangkat
- Model subscription (Login/Coba Sekarang)
- Optimasi mobile & keamanan API key di backend

## Outcome

- **10K+ pengguna aktif** dan **1M+ pesan terkirim** dengan **99.9% uptime** — angka ini diklaim di halaman produk (per Juli 2026), belum diverifikasi independen oleh pihak ketiga.
